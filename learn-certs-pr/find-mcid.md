# <a name="find-your-mcid"></a>尋找您的 MCID

若要尋找您的 Microsoft 認證識別碼 (MCID) ，請登入您的 [認證儀表板。](https://aka.ms/certdashboard) 您的 MCID 位於頁面右上角。

![MCID 位於儀表板右上角](images/certification-dashboard-find-mcid.png)
